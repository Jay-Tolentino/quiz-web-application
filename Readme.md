# Quiz Web Application

A lightweight web based quiz platform built using HTML, CSS, and JavaScript. The application supports user authentication, quiz management, and timed assessments, with optional Firebase integration.

Developed in a collaborative software engineering course, I served as a primary technical contributor across core application features and testing.

---

## Overview

The system includes:

- User login and authentication flow
- Timed quiz functionality for students
- Admin dashboard for managing quiz content
- Modular client side JavaScript architecture
- Optional Firebase backend integration

## Architecture Overview

Project Type: Static Web Application (HTML / CSS / JavaScript)

Core structure:

- Root HTML pages manage user and admin flows
- Client side JavaScript modules handle quiz logic and UI behavior
- Domain models manage quiz data and scoring
- FirebaseHandler.js supports optional backend integration
- CSS manages global styling
- images and fonts contain static assets

Key JavaScript modules:

- quizManagement.js – Quiz creation and management
- quizAppletScript.js – Quiz runtime logic
- quizTemplateScript.js – Template rendering
- StudentScore.js, Quiz.js, Item.js – Domain models
